Understanding C++

What is C++?

C++ is one of the most popular and advanced programming languages used by professionals. The language is renowned for its difficulty but also its multi-paradigm nature and advanced syntax.

Why is C++ so important?

C++ is used almost everywhere in the digital world! Some examples of C++ uses would be making browsers, applications and software.

What other uses does C++ have?

Although C++ is a very popular choice for the building of browsers and software, its most popular use is building operating systems. Some examples being Mac Os, Windows and Linux.

What is a web developer?

A Web Developer is in charge of ensuring websites look good and function properly. Web developers collaborate with website and graphic designers, monitor website traffic, troubleshoot website problems when they arise, and update websites as necessary.

What are the average salaries?

Salaries for junior or entry-level web developers can range from £19,000 to £25,000. Mid-level and senior web developers usually earn between £25,000 and £35,000. Lead developers typically earn between £35,000 and £60,000, potentially rising to £75,000 in London.

Do I need to know any other languages to start learning C++?

Because C++ is a more technically advanced language compared to the likes of Javascript, yes, it is recommended you know some programming before jumping into the waters of C++.

What jobs use C++?

An example of a job that uses C++ as a core language is a game developer. C++ is normally used to make the game systems.
