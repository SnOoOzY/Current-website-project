Understanding Javascript

What is Javascript?

Javascript is a core coding language in the World Wide Web, along with HTML and CSS. Around 98% of websites use Javascript to focus on the webpage behaviour aspects.

Why is Javascript so important?

Javascript is essential in all internet experiences. It is able to create more interaction and complexity into their applications. Search engines, ecommerce, content management systems, responsive design, social media and phone apps would not be possible features without Javascript.

Can Javascript be used in more ways than webpage building?

Yes, Javascript can be used in more ways that do not include webpages. Javascript does also play a role in databases and other content management systems. Although, a less popular use of Javascript would be game design. The vast majority of games which do use Javascript are web based games, a massive exception to this however being the hit game Minecraft.

What is a web developer?

A Web Developer is in charge of ensuring websites look good and function properly. Web developers collaborate with website and graphic designers, monitor website traffic, troubleshoot website problems when they arise, and update websites as necessary.

What are the average salaries?

Salaries for junior or entry-level web developers can range from £19,000 to £25,000. Mid-level and senior web developers usually earn between £25,000 and £35,000. Lead developers typically earn between £35,000 and £60,000, potentially rising to £75,000 in London.

Do I need to know any other languages to start learning Javascript?

The short answer to that is no, you do not need to know any other languages to start Javascript. Javascript is actually one of the more easier languages to start off with for beginners and can get early programmers off to a good start.

What jobs use JavaScript?

The most common jobs that use JavaScript are web developers / game developers.
